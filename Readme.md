# Colored Lasers Pointer for World of Tanks (Lesta Games)

[![Version](https://img.shields.io/badge/version-8.3.70-blue)]()
[![Game version](https://img.shields.io/badge/game-1.42.0.0-green)]()
[![License](https://img.shields.io/badge/license-All%20rights%20reserved-red)]()

Adds 3D laser beams to enemy tanks. Beam color indicates shots to kill:
- 🔴 **Red** – 1–2 shots
- 🟡 **Yellow** – 3–8 shots
- 🟢 **Green** – >8 shots

## Installation
1. Download `ru.DevGeek.IT.ColoredLasersPointer.mtmod`
2. Copy to `World_of_Tanks/mods/1.42.0.0/`
3. Launch the game

## Controls
- `NumPad 3` – toggle mod on/off
- `NumPad 7` – toggle colored mode

## Configuration
Edit `mods/configs/ColoredLasersPointer/mod_ColoredLasersPointer.json`
