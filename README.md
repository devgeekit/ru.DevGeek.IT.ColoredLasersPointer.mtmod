# ru.DevGeek.IT.ColoredLasersPointer.mtmod
3D-лазеры на вражеские танки. Цвет луча зависит от выстрелов до уничтожения: красный (1–2), жёлтый (3–8), зелёный (9+). Управление: NumPad 3 – вкл/выкл мода, NumPad 7 – цветной режим (все лучи красные). Поддерживает фильтр по типам техники, комбинации клавиш. Конфиг JSON (mods/configs/ColoredLasersPointer). Lesta Games 1.42.
